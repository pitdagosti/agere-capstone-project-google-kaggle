ROLE 1: MOBILE APP DEVELOPER
===============================================================

JOB DESCRIPTION:
- Develop native Android applications using Kotlin/Java in Android Studio
- Build cross-platform applications using .NET MAUI (C#) in Visual Studio
- Collaborate with clients to understand requirements and deliver MVPs
- Implement responsive UI/UX designs and ensure app performance
- Write clean, maintainable code with proper documentation
- Participate in code reviews and testing processes

REQUIRED SKILLS:
- Proficiency in Kotlin/Java or C# (.NET MAUI)
- Experience with Android Studio or Visual Studio
- Understanding of mobile UI/UX principles
- Knowledge of RESTful APIs and backend integration
- Version control with Git
- Problem-solving and debugging skills

  - Alignment with startup culture: 10 points

ROLE 3: AI/ML ENGINEER (GENERATIVE AI SOLUTIONS)
===============================================================

JOB DESCRIPTION:
- Design and develop custom knowledge-base chatbots using LLMs
- Build AI-powered workflow automation agents
- Integrate AI solutions with client systems and databases
- Fine-tune and optimize AI models for specific use cases
- Implement RAG (Retrieval-Augmented Generation) systems
- Stay updated with latest AI/ML trends and tools

REQUIRED SKILLS:
- Experience with OpenAI API, Anthropic Claude, or similar LLM APIs
- Python programming for AI/ML
- Knowledge of vector databases (Pinecone, ChromaDB, Weaviate)

- API development and integration
- Familiarity with LangChain, LlamaIndex, or similar frameworks
